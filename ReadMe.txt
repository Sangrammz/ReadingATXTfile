Reading  A Txt File from local desktop server
-----------------------------------------------


This is a very easy process to create Bot to read data from your local computer,


How To Create
------------

1.The Main Goal is to find the txt file from the local computer and display it via the bot

2.Now First seach for "read text file activity"drag and drop to your sequence

3.Drag a "logMessage" activity to viw the log files.

4.And then deag a MessageBox to Display the message.

5.Set the file path inside a "File path/ Name"

6.For Best Practice place the file inside the project folder to Avoid data misplace.

7.now create a variable to get the data as a output

8.now place the variable in "logmessage" activity and "messageBox" activity to display the message

9.Done!. Now Run the file , you can see the data inside the txt file .
